cssBreak.js

A cross-browser CSS-detection module for breakpoints.

The following breakpoints are set:

  // Add media queries
  // cssBreak.add(breakpointName, mediaQuery, isIe:=false)
  cssBreak.add('small', '(min-width: 500px)')
  cssBreak.add('medium', '(min-width: 800px)', true) // Etc...

Which causes events to be fired on the window object: 

  cssBreak.on('enter:small', function () {
    console.log('Entering small breakpoint')
  })

  cssBreak.on('exit:small', function () {
    console.log('Exiting small breakpoint')
  }) // Etc...

Therefore, the exit:mq event is called when you are leaving the breakpoint for a smaller size, and the enter:mq event called when you reach the breakpoint.

Look in the console.log to see the results.
